5. Questions and Answers

  1. Q: Where does the name "baduki" come from?

    A: Baduk means Go in Korean.

  2. Q: Where do I find the rules of Go?

    A: Check the Documentation folder included with this package, you will find the file Rules and also some other documents for beginners.

  3. Q: Where can I find more information about Computer Go?

    A: The Gnu Go homepage is a good starting point.
