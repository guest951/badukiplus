This Handbook describes Badukiplus Version 0.1. Badukiplus is a simple program to play Go, rewritten in C++ from Baduki, in turn based on AmiGo, in turn based on the original Pascal program by Stoney Ballard dating from 1983. BadukiPlus is similar to Wallyplus, another Go engine rewrite to C++.
